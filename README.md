

## YAHTZEE (PyQT Version)

A fully playable 2-player game of Yahtzee, built using Python and PySide6(QT).

Please see the program's "View Instructions" page for details on rules.

-----

### Author

  * Oliver Osmond

-----

### Requirements

  * Python 3.8+
  * PySide6\>=6

-----

### Installation

**Step 1:** Please clone from `https://github.com/meeses-o/yhatzee_gui.git`

**Step 2:** Create and activate a working virtual environment (venv):

```bash
python3 -m venv venv
```

  * **For macOS/Linux:**
    ```bash
    source venv/bin/activate
    ```
  * **For Windows:**
    ```bash
    venv\Scripts\activate
    ```

-----

### Running the Game

Please run the following file to play:

```bash
interface/page_1.py
```
